## Utilisation du modèle sur Kaggle

Pour reproduire les résultats, exécutez le code sur Kaggle :

1. **Créez un notebook** sur [Kaggle](https://www.kaggle.com/)
2. **Ajoutez la base de données** Task10_Colon en entrée
3. **Copiez-collez le code** `modele_segmentation.py`
4. **Exécutez toutes les cellules**

Le code générera automatiquement les résultats dans le dossier de sortie.
